Wellcome to README file!

--> Introduction
    The Weather Web App is a simple and intuitive application that provides current weather information for any city in the world. Users can search for a city and get details like temperature, humidity, wind speed, and weather conditions.

--> Features
    Search for current weather by city name.
    Display temperature, humidity, wind speed, and weather conditions.
    Responsive design for optimal viewing on any device.

--> Technologies Used
    Frontend:

    HTML5
    CSS3
    JavaScript
    API: OpenWeatherMap API
    Online Tools : Tailwind CSS, boxicons, font-awesome, outube for learning, additional features.
